How We Rank

METHODOLOGY & TRANSPARENCY
🧮
Value Score Formula
Every model receives a Value Score computed from four metrics. Free models get the highest scores because cost approaches zero.
V = (Intel×33 + Speed×25 + Stab×25) / (Cost + ε)
Where ε = 0.001 prevents division by zero for free models. Higher score = better value for the user.
🏷️
Tier Legend
🔑
PRIMARY (OAuth Bridge) — Top-tier models accessed via existing subscriptions (ChatGPT Plus, Claude.ai free tier, AI Studio). Zero extra cost.
🆓
FREE (Free Frontier) — Genuinely free API models via OpenRouter. No subscription needed. Includes daily-limit and community-hosted models.
⚔️
RESCUE (Heavy Hitter) — Paid models used only as last-resort when all free options fail. Deploys, solves, then immediately reverts to free.
📊
Metric Scale (0–100)
🧠
Intelligence — Composite of reasoning, coding, and general knowledge benchmarks. Based on OpenRouter performance data and cross-referenced with public benchmarks.
⚡
Speed — Output tokens per second. 99 = fastest tier (>100 tok/s), 50 = moderate (∼30 tok/s).
🛡️
Stability — API uptime and reliability. 98 = near-perfect availability, <80 = frequent outages or rate limits.
💰
Cost — Price per output token averaged across providers. $0.00 = free access.
📡
Data Sources

🔜
LMArena (Chatbot Arena) — Crowdsourced ELO rankings. Integration planned for cross-validation.
🔜
Artificial Analysis — Independent benchmark aggregator. Integration planned for intelligence index cross-referencing.
⚠ Scores are normalized estimates. For absolute benchmark data, we recommend cross-referencing with artificialanalysis.ai and lmarena.ai.

About AIchain

🧬 What is AIchain?

AIchain is an open-source, decentralized intelligence routing layer that ranks every accessible AI model by real value — intelligence, speed, stability, and cost. It was built for users who want maximum AI at zero cost, not for providers who want to sell subscriptions.

The routing table is automatically generated and synced every 6 hours via the AIchain Bridge, which pulls data from OpenRouter and normalizes it into a universal value ranking.
